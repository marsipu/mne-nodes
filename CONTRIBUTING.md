# Contributing to mne-nodes

Contributions are very welcome! Thank you for taking the time to contribute to mne-nodes.

mne-nodes is supposed to be a **GUI framework** for building and running MEG/EEG workflows.
It should not contain the any analysis logic (core-functions being the exception during early development).
It should be implemented in separate, importable Python modules/packages and maintained in their own repositories (for example, on GitHub).
